- **index.html** - The main HTML file with all local references
- **css/** - All CSS stylesheets

- **js/** - All JavaScript files

  - `webflow.schunk.2.js` - Webflow framework script 2
  - `webflow.455b30af.js` - Webflow framework script 3

## Usage

Simply open `index.html` in any modern web browser. All resources are local and the page should render identically to the original website.
